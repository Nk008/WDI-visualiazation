## How to Run

### Option 1 — Google Colab (Recommended)
1. Click the notebook file above
2. Click **Open in Colab** button
3. Run all cells: **Runtime → Run all**

### Option 2 — Local Installation
```bash
git clone https://github.com/YOUR_USERNAME/LDS7004M-DataVisualisation.git
cd LDS7004M-DataVisualisation
pip install pandas matplotlib seaborn plotly azure-storage-blob
jupyter notebook LDS7004M_Data_Visualisation.ipynb
```
